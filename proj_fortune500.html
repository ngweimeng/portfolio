Fortune 500 Companies Bar Chart Race

Visualising rank and revenue of Fortune 500 Companies in a Bar Chart Race using Python.

Bar Chart Races in Python quite easily using the matplotlib library. In order to streamline the process with fewer lines of code, I utilized the bar-chart-race package, which is built on top of matplotlib. The official documentation for this package can be found here.
